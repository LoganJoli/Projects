# Cybersecurity
This project that I have completed was a security audit for an imanginary company given to me in my google cybersecurity certificate. In this project I was given a security audit that was completed and using the security audit, I was tasked with a controls and compliance checklist that I had to complete. After completing the audit I was finally tasked with providing recommendations, related to controls and/or compliance needs, that the imaginary IT manager could communicate to stakeholders to reduce risks to assets and improve imaginary companies security posture.
